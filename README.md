**LuxeTravels – Travel Companion Web Application**

This project is a travel companion web application built using HTML, CSS, and JavaScript. It helps users plan trips efficiently by providing features like destination exploration, weather updates, wishlist management, and trip booking all in one place.

**Features**

•	User Authentication: Register and login system using LocalStorage
•	Travel Wishlist: Add, update, and delete travel plans
•	Trip Booking: Book trips with destination, dates, and number of travellers
•	Weather Information: Get real-time weather updates for any city
•	Interactive UI: Image sliders, animations, and responsive design
•	Navigation System: Dropdown menus and dynamic UI updates

**Prerequisites**

1.	A modern web browser (Chrome, Edge, Firefox, etc.)
2.	Internet connection (for weather API)

**Installation**

1.	Clone the repository: git clone <repository_url> cd LuxeTravels
2.	Open the project: Open homepage.html in your browser

**Usage**

•	Open the application in your browser
•	Register a new account or login
•	Add trips to your wishlist
•	Book a trip
•	Check weather for destinations
•	Navigate through different pages using the menu

**Tech Stack**

1.	Frontend: HTML, CSS, JavaScript
2.	Storage: LocalStorage
3.	API: OpenWeatherMap API

**API Endpoints**

•	Weather data is fetched using OpenWeatherMap API

**Key Functional Pages**

1.	Homepage: Navigation, sliders, and travel sections
2.	Login/Register: User authentication with validation
3.	Booking Page: Trip planning and confirmation
4.	Wishlist: Manage travel plans (CRUD operations)
5.	Weather Page: Real-time weather display
6.	About Page: Information about the platform
