**LuxeTravels – Travel Companion Web Application**

This project is a travel companion web application built using HTML, CSS, and JavaScript. It helps users plan trips efficiently by providing 
features like destination exploration, weather updates, wishlist management, and trip booking—all in one place.

**Features**

User Authentication: Register and login system using LocalStorage
Travel Wishlist: Add, update, and delete travel plans
Trip Booking: Book trips with destination, dates, and number of travelers
Weather Information: Get real-time weather updates for any city
Interactive UI: Image sliders, animations, and responsive design
Navigation System: Dropdown menus and dynamic UI updates

**Prerequisites**

A modern web browser (Chrome, Edge, Firefox, etc.)
Internet connection (for weather API)

**Installation**

Clone the repository:git clone <repository_url> cd LuxeTravels
Open the project: Open homepage.html in your browser

**Usage**

Open the application in your browser
Register a new account or login
Explore features:
Add trips to your wishlist
Book a trip
Check weather for destinations
Navigate through different pages using the menu

**Tech Stack**

Frontend: HTML, CSS, JavaScript
Storage: LocalStorage
API: OpenWeatherMap API

**API Endpoints**

Weather data is fetched using OpenWeatherMap API

**Key Functional Pages**

Homepage: Navigation, sliders, and travel sections
Login/Register: User authentication with validation
Booking Page: Trip planning and confirmation
Wishlist: Manage travel plans (CRUD operations)
Weather Page: Real-time weather display
About Page: Information about the platform
